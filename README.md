## Getting Started

### Prerequisites

- Git
- [Python >= 3.12](https://www.python.org/downloads/)
- [Poetry >= 2](https://python-poetry.org/)

### Set up Environment

- Configure access to Poetry virtual environment

  ```
  poetry config virtualenvs.in-project true
  ```

- Install dependencies

  ```
  poetry install
  ```
